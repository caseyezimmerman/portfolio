City Runner
Github Repository
A web app for runners that generates running loops around them based on the user's current location and desired distance in miles. The app renders 4 maps for the user to choose from and then their data (time and distance) is displayed on a results page when they finish their run.

Pygame
Github Repository
A soccer game I made using Python that helped us get familiar with object oriented programming. The goal of this game is to get to the soccer ball before your opponent does. The more points you get, the more opponents you will have. You can also throw cups at your opponents to freeze them for 3 seconds.
